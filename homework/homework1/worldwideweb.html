The Internet

Main World Wide Web Email Wi-Fi

Computer Scientist, Tim Berners-Lee was the inventor of the World Wide Web after the online world became recognized in 1990. The web was created to allow access to data online from websites and hyperlinks. The world wide web is what made the Internet popular among the public, which continues to serve a crucial step in accessing information that we now do on a daily basis.

The World Wide Web contains billions of documents. It uses the Internet as its transport mechanism to share its collections of documents, such as web pages.

    The server is responsible for handling the request sent from the client.

    The client sends the user request to a WWW server, directing them to the Hypertext Transfer Protocol (HTTP)
